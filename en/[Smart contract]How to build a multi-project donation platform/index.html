安装ethers

安装 ethereum-waffle

运行 npx hardhat 并选择（▸ Create an empty hardhat.config.js）创建hardhat.config.js文件

打开 config.js

创建contracts文件夹

创建test文件夹

创建scripts文件夹

在contract目录下创建 WaterDrop.sol

在scripts 文件夹下创建WaterDrop-deploy.js

执行编译命令
